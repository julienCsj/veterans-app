Le calendrier
Cet espace vous permet de plannifier vos futures missions ou entrainement. Afin de faire apparaitre le message proposant aux joueurs de s'inscrire à la mission, vous devez, lors de la création de l'évènement, renseigner l'URL du post de votre mission sur le forum.
Procédure de création d'une mission :
  • Comme avant, il faut créer un sujet sur le forum
  • Se rendre sur le calendrier, et créer un évènement (bien renseigner l'URL du sujet dans le formulaire de création de l'évènement)
  • Placer l'évènement sur le calendrier à la date voulue
  • Le message proposant de s'inscrire sur le forum est visible
Il est également possible de créer des évènements prévisionnels. Ces évènements servent à se mettre d'accord sur une date. Il ne seront pas proposé sur le forum tant que l'évènement n'est pas validé.
